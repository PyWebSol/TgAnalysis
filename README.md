# PyWeb TgAnalysis

<a href="https://t.me/PyWebChan"><img src="https://img.shields.io/badge/follow-@PyWebChan-1DA1F2?logo=telegram&style=%7Bstyle%7D"></a>

Простой инструмент для анализирования телеграм чата.

## Установка:
```bash
pip install -r requirements.txt
```

## Запуск:
```bash
python main.py
```

### Результат выводится в консоль и записывается в файл result.txt
